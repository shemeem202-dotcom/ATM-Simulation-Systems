# ATM-Simulation-Systems

ATM Simulation System (Python)

A simple and interactive ATM Simulation System built using Python. This project demonstrates basic banking operations such as authentication, balance enquiry, cash deposit, cash withdrawal, transaction history, and PIN change using a command-line interface.

 Features

 PIN-based user authentication

 Balance enquiry

 Cash deposit

 Cash withdrawal

 Transaction history

 Change ATM PIN

 Menu-driven interface

 Technologies Used

Python 3

Command Line Interface (CLI)

 How to Run the Program
1. Clone the Repository
git clone https://github.com/your-username/atm-simulation-system.git

2. Navigate to the Project Folder
cd atm-simulation-system

3. Run the Program
python atm.py


Make sure Python 3 is installed on your system.
